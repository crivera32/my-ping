# my-ping
Network diagnostic tool
Runs on Linux Ubuntu
To compile, run "make myping"
Usage:           ./myping --di <dst_ip>
Optional Flags:  --si <src_ip> 
                 --de <dst_mac>
                 --se <src_mac>
                 --dp <dst_port>
                 --sp <src_port>
                 --ttl <time_to_live>
                 --rst <rst_mode>
                 --interface <network_interface_name>
                 --devind <network_interface_index>
